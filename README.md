# 图书借阅管理系统

#### 介绍

Spring Boot + MyBatis + thymeleaf + MySQL + jQuery + ajax等实现轻便的图书借阅管理系统

#### 软件架构

开发环境：IDEA + JDK1.8 + Navicat + Maven


#### 安装教程

1.  导入项目，刷新Maven依赖。
2.  更改yml文件中数据库连接配置
3.  导入数据库表结构与数据

#### 使用说明

1.  登录用户名admin，密码1
